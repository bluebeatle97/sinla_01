## 🚀 개선사항
</br>

<b>1. 다양한 라이브러리 활용</b>: 슬라이드 로직을 바닐라 JS로 설계함, 다음 프로젝트에선 `Swiper` 같은 라이브러리를 적절하게 사용해볼 예정임.

<b>2. 더 효율적인 프레임워크 사용</b>: JavaScript와 jQuery만으로 동적인 DOM 조작과 상태 관리를 하다 보면 코드가 복잡해지고 유지보수가 어려워질 수 있음 컴포넌트 단위의 재사용성 확보와 체계적인 상태 관리를 위해 다음 프로젝트에서는 React를 도입해 보겠음.

<b>3. 웹 성능 최적화</b>: 쇼핑몰 사이트 특성상 다수의 상품 이미지를 렌더링해야 하므로 초기 로딩 속도가 저하될 우려가 있음. 이를 개선하기 위해 다음 프로젝트에서는 화면에 노출될 때만 이미지를 불러오는 지연 로딩 기법이나 WebP 이미지 포맷 등을 적용하여 프론트엔드 렌더링 성능을 최적화해볼 예정임.
